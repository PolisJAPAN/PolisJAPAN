FEATURE

Polisの特徴

#1

反論ではなく、
新たな意見を投稿

#Polisでは、他者の意見に対して返信を送ることはできません。直接反論を送るのではなく、「新たな視点」を投稿します。お互いの立場への理解が深まり、不毛な議論、いわゆるレスバ状態を回避します。

#2

極論ではなく、
“合意できる意見”
にフォーカス

Polisでは、バズらせるための極論には注目が集まりません。それぞれの立場の人々から、最も多く同意された”落とし所”となる意見がフォーカスされます。
これまでのSNSで注目されなかった、中間層の意見が見える化されます。

#3

選んだ意見で、
自分の立ち位置を可視化

同意した意見を元に、あなたがどの立ち位置にいるのかを可視化します。また、他の回答者の分布もわかるため、自己理解と他者理解を深めることができます。

まずは体験してみましょう！

このサイトについて

#

まず、Polisってなに？

Polisは、The Computational Democracy Project社が開発・提供しているオープンソースプラットフォームです。参加者の意見を短文で投稿・投票し合うことで、多様な視点をリアルタイムに収集・分析できるオープンソースの会話プラットフォームです。機械学習を用いたクラスタリングにより、意見の分布や合意点が可視化され、大規模な合意形成や相互理解を促進します。

リンク

#

PolisとPolisJAPANの違いは？

PolisJAPANは日本へのPolisの普及を目的としたプロジェクトです。
Polisのオープンソースライセンス(AGPL3)を継承し、日本でのユースケースに合わせたUI / UXを提供することで日本のユーザーに親しみやすい形でPolisの仕組みを提供します。
Polisの普及を志す有志メンバーにて運営しています。

#

PolisJAPANは営利目的ですか？

いいえ。PolisJAPANは営利目的ではありません。
Polisを日本へ普及することを目的とした有志プロジェクトです。

※ 現在はサーバー料金についても有志の自費にて運営しておりますが、
アクセス数増加により賄いきれなくなった場合は、サービス維持のためのアドセンスプログラムを
導入する可能性があります。
